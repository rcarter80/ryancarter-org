Teaching materials for tenure file

Purpose of this site

Interfolio does not support media files like MP3s, so I can't upload certain teaching materials like recordings of student compositions. This site summarizes all of my teaching materials, identifying which files can be downloaded from Interfolio and which can be found through this page (either by navigating to external links or listening to recordings hosted here.)

Music 270 - INTRODUCTION TO MUSIC, SOUND, AND TECHNOLOGY

Included on Interfolio
MUS270-Syllabus.pdf
Not included on Interfolio
Kayla Beardslee's Project #1
The prompt for this assignment is to create a composition in any musical style using only recorded audio. (Students are allowed to combine recordings they have made with extant recordings, as long as they cite their sources for the latter.)
Anthony Christiana’s final project
The prompt for this project is wide open. It simply needs to: 1) involve sound, 2) involve computers, 3) be shareable on the Internet.
"Drone on Drone" project
Ella Gant had the idea of a collaboration between her students in Advanced Video and my students in Introduction to Music, Sound, and Technology. You can read about the project at the link above.
