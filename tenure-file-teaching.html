Chamber Works

Chamber Works is my first portrait album, available on KAIROS Records. You can also hear it on Spotify, Tidal, and other streaming platforms. Scores for each track are available in the "Scores for 1st album" section on Interfolio; some of these works were composed before I arrived at Hamilton. A Patient Resolve

This is an album of algorithmically generated music that I composed, which I perform under the stage name Headless Monkey Attack. The music is implemented in the RTcmix programming language embedded in the Max/MSP programming environment; there are also some sound files that I recorded, which are triggered by the same RTcmix scripts that handle the rest of the audio. I use Jitter (and shaders I wrote in GLSL) for live video processing. I control all of the interactive elements with a Gametrak "tether" controller. For more information, visit: headlessmonkeyattack.com
